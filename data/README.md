# data/

Small, public datasets used in notes, activities, and assignments.

Rules:

- **Public only.** No student data, no rosters, no anything pulled from
  a school system. If a dataset came from a student or course, do not
  put it here.
- **Cleaned and small.** Each file should be readable, well-named, and
  small enough to ship with the site. Larger raw datasets stay in a
  research repo.
- **Document the source.** Each dataset should have a short comment or
  sidecar describing where it came from and any cleaning that was done.
